# Campus Notification System

## Project Overview

This project is a Campus Notification System developed using React.js for frontend and Node.js for backend proxy handling.

The application fetches notifications from the evaluation API and displays them in a professional responsive UI.

---

# Features

* Fetch notifications from API
* Priority-based notification display
* Responsive modern UI
* Backend proxy server for secure API handling
* Logging middleware integration
* React frontend using Vite
* Express backend server

---

# Tech Stack

## Frontend

* React.js
* Vite
* CSS

## Backend

* Node.js
* Express.js
* Axios
* CORS
