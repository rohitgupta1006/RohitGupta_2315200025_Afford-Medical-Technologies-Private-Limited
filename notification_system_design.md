# Notification System Design

## Overview

This project is a Campus Notification System.

The frontend fetches notifications from the backend and displays them in a responsive UI.

The backend securely communicates with the external notifications API.

---

# Architecture

Frontend:

* React.js
* Vite
* CSS

Backend:

* Node.js
* Express.js
* Axios

---

# Flow

Frontend → Backend → External Notification API

---

# Features

* Fetch notifications
* Responsive design
* Notification cards
* API integration
* Error handling
* Secure backend proxy
